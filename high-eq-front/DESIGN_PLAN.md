# 高情商回复生成助手 - 网站设计方案

## 设计理念

**设计方向**: 现代科技 + 人文温暖
- 融合科技的精准感与人文的温度
- 展现 AI 赋能下的智能化，同时强调情商、人性化的核心价值

## 核心设计原则

1. **清晰的信息层级**: 通过视觉权重引导用户理解产品的核心价值
2. **交互式体验**: 使用演示卡片、动画、渐进式展示来增强用户参与感
3. **现代极简**: 避免过度装饰，用空间、颜色、排版传达信息
4. **响应式设计**: 完美适配移动端和桌面端

## 色彩哲学

- **主色渐变**: 蓝色 (#3b82f6) → 紫色 (#8b5cf6) → 粉色 (#ec4899) - 营造现代感和温暖感
- **辅助色**: 翠绿 (#10b981) - 代表成功、积极、情商
- **中性色**: 灰度系统 - 保持清晰度和可读性
- **背景**: 浅色系渐变 (蓝色50 → 紫色50 → 粉色50) - 现代、清爽
- **文字**: 深灰 (#1f2937) - 保证可读性

## 布局范式

### 首页 (Home.tsx)
- **导航栏**: 固定顶部，毛玻璃效果，Logo + CTA 按钮
- **Hero Section**:
  - 大标题（渐变文字）
  - 副标题（产品定位）
  - CTA 按钮（主按钮 + 次要按钮）
  - 浮动装饰元素（渐变光晕）
  - 波动背景纹理
- **演示区域**: 可切换的场景演示卡片（同事/朋友/伴侣）
- **角色展示**: 6 种角色卡片（emoji + 渐变文字）
- **语气展示**: 5 种语气卡片
- **功能特点**: 3 个特性卡片（AI 驱动、情商优先、随时回顾）
- **CTA 区域**: 彩色渐变背景 + 大标题
- **Footer**: 简洁的版权信息

## 签名元素

1. **渐变背景**: 蓝紫粉三色渐变
2. **圆角卡片**: 统一的圆角设计（rounded-xl）
3. **浮动装饰**: 脉冲式渐变光晕效果
4. **Emoji 图标**: 增加趣味性和亲和力
5. **微动画**:
   - 卡片悬停上浮 + 阴影增强
   - 标签弹跳动画
   - Emoji 缩放效果

## 交互哲学

- 卡片悬停时有阴影提升 + 轻微上浮
- 按钮按下有明确的反馈
- 场景切换有平滑过渡
- 渐变文字增强视觉吸引力

## 动画指南

- 过渡时间: 200-300ms (快速反应)
- 缓动函数: ease-in-out (自然流畅)
- 悬停效果: transform hover:-translate-y-1
- 浮动效果: animate-pulse (装饰元素)

## 排版系统

- **标题字体**: Noto Sans SC (中文) + Poppins (英文)
- **正文字体**: Noto Sans SC - 清晰易读，支持中文
- **标题层级**:
  - H1: 48-72px, 700 weight (渐变文字)
  - H2: 36-48px, 600 weight
  - H3: 24px, 600 weight
  - Body: 16px, 400 weight
  - Small: 14px, 400 weight

## 页面结构

1. **导航栏**: 固定顶部，Logo + 开始使用按钮
2. **Hero Section**: 产品标语 + 演示按钮 + 信任指标
3. **演示区域**: 可切换场景的交互式演示
4. **角色展示**: 6 种预设角色
5. **语气展示**: 5 种语气风格
6. **功能特点**: 3 个核心优势
7. **CTA 区域**: 行动号召
8. **Footer**: 简洁版权信息

## 组件库

- **Button**: Radix UI Button + Tailwind 样式
- **Card**: shadcn/ui Card 组件
- **Toast**: Sonner 通知组件
- **Icon**: lucide-react 图标库

## 响应式断点

- **sm**: 640px（移动端）
- **md**: 768px（平板）
- **lg**: 1024px（桌面）
- **xl**: 1280px（大屏）

## Dark Mode 支持

所有页面都支持 Dark Mode，使用 Tailwind 的 `dark:` 前缀。
